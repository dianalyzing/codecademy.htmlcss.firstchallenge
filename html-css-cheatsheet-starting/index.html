Learn HTML: Elements and Structure

What is HTML?

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

What are HTML Elements?

An HTML element is defined by a start tag, some content, and an end tag. The content of an HTML element is the information between the opening and closing tags of an element.

What is HTML Structure?

HTML describes the structure of a website. When written, HTML is organized into a family tree structure. HTML elements can have parents, grandparents, siblings, children, grandchildren, etc.

Here is your guide to HTML Element and Structure tags.

HTML Elements and Structure Tags

Tag Name Description
li List Item Element The list item element create list items in ordered and unordered lists.
video Video Element The element embeds a media player for video playback.
em Emphasis Element The emphasis element emphasizes text and browsers will usually italicize the emphasized text by default.
ol Ordered List Element The ordered list element creates a list of items in sequential order.
div Div Element The div element is used as a container that divides an HTML document into sections and is short for “division”.
br Line Break Element The line break element will create a line break in text.
img Image Element HTML image elements embed images in documents.
h1 - h6 Heading Elements HTML can use six different levels of heading elements. The heading elements are ordered from the highest level, h1, to the lowest level, h6.
p Paragraph Element The paragraph element contains and displays a block of text.
ul Unordered List Element The unordered list element is used to create a list of items in no particular order.
body Body Element The element represents the content of an HTML document.
span Span Element The element is an inline container for text and can be used to group text for styling purposes.
strong Strong Element The element highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.
a Anchor Element The anchor element is used to create hyperlinks in an HTML document.
head Head Element The element contains general information about an HTML page that isn't displayed on the page itself.
html HTML Element The html element, the root of an HTML document, should be added after the DOCTYPE declaration. All content/structure for an HTML document should be contained between the opening and closing html tags.
title Title Element The title element contains a text that defines the title of an HTML document. The title is displayed in the browser's title bar or tab in which the HTML page is displayed. The title element can only be contained inside a document's head element.
